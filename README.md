# Data-Centric-AI-Lab7

## Overview
This repository contains Lab 7 experiments comparing Data-Centric AI and Model-Centric AI approaches. The purpose of this lab is to understand how improving data quality versus improving model architecture impacts overall machine learning performance.

## Objectives
- Understand the difference between Data-Centric AI and Model-Centric AI
- Train a baseline machine learning model
- Improve dataset quality through preprocessing and cleaning
- Improve model performance through tuning and optimization
- Compare and analyze results from both approaches

## Methodology
1. Train a baseline model.
2. Apply data preprocessing techniques such as cleaning, handling missing values, and normalization.
3. Retrain the model using improved data (Data-Centric approach).
4. Tune and optimize the model (Model-Centric approach).
5. Evaluate and compare performance metrics.
